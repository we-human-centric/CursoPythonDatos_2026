Día 3 · Fundamentos de Python II – Control, colecciones y funciones

Fecha: Miércoles 23 de abril de 2026 · Duración: 2 horas

Introducción

Hoy damos el salto de scripts lineales a código organizado. Controlamos el flujo con condicionales y bucles, almacenamos datos en estructuras (listas, tuplas, sets, diccionarios), y modularizamos con funciones. Esta sesión es clave para todo lo que viene.

Contenidos de la sesión

1. Condicionales

if / elif / else. Ejecutan código solo si se cumple una condición. Expresiones ternarias: x if condicion else y.

2. Bucles

for itera sobre colecciones. while ejecuta mientras una condición sea verdad. break sale del bucle, continue salta a la próxima iteración.

3. Estructuras de datos

  • Listas: mutables, ordenadas, con duplicados. [1, 2, 3]
  • Tuplas: inmutables, ordenadas. (1, 2, 3)
  • Sets: sin duplicados, operaciones de conjuntos. {1, 2, 3}
  • Diccionarios: pares clave-valor, búsqueda O(1). {"nombre": "Juan"}

4. Comprensiones

List comprehensions: [x*2 for x in xs if x > 0]. Más legibles y rápidas que bucles equivalentes.

5. Funciones

Encapsulan lógica. Parámetros posicionales, por nombre, valores por defecto. *args, **kwargs. Type hints y docstrings. Principio: una función, una responsabilidad.

6. Manejo de errores

try / except / else / finally. Captura excepciones específicas, no genéricas.
