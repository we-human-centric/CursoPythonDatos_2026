📂 Casos de Estudio

Proyectos del Seminario  ·  Grupos de 3 estudiantes  ·  Entrega 15 de mayo 2026

Cada caso propone un problema analítico real organizado en 5 fases pedagógicas progresivas: comprensión del dominio → exploración de datos → modelado → producto analítico → comunicación de hallazgos. El grupo elige un solo caso y lo desarrolla durante las 4 semanas del seminario.

Flujo de trabajo: leer el caso completo antes de descargar datos  ·  formular hipótesis en la Fase 1 antes de abrir archivos  ·  comparar siempre ≥ 2 modelos con métricas apropiadas  ·  entregar API + Dashboard funcionales en la Semana 4.
