Repositorio oficial del **Seminario de Análisis Exploratorio de Datos**. En 17 sesiones y 4 semanas los estudiantes construyen un **Producto de Software Analítico** completo: desde la pregunta de negocio hasta un dashboard en Streamlit y una API en FastAPI.

---

## 🚀 Cómo usar este repositorio

### Opción A · Google Colab (sin instalar nada)
Haz clic en el botón **Open in Colab** de cada sesión. Solo necesitas una cuenta de Google.

### Opción B · Ejecución local
```bash
git clone https://github.com/we-human-centric/CursoPythonDatos_2026.git
cd CursoPythonDatos_2026
python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
jupyter lab
```
