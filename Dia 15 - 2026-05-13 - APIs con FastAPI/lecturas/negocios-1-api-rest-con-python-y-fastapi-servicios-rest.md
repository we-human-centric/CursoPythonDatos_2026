API REST con Python y FastAPI

El curso API REST con Python y FastAPI está orientado a aprender a crear servicios web REST utilizando Python y el framework FastAPI. A lo largo de la formación se trabajan los fundamentos de la arquitectura de servicios web y REST, junto con la preparación del entorno de trabajo necesario para el desarrollo de APIs.
El contenido cubre la implementación de servicios REST simples y completos, el uso de FastAPI y uvicorn, el testeo de servicios con Postman y la documentación de endpoints mediante Swagger. También se abordan los códigos de estado, la gestión de errores y el desarrollo de un servicio REST CRUD aplicado a un caso práctico.


Bases de datos, seguridad y despliegue de servicios REST

La formación avanza con la integración de servicios REST con bases de datos relacionales desde Python, utilizando mysql-connector, y la implementación de 
operaciones CRUD conectadas a base de datos. 
Asimismo, se trabaja el consumo de servicios REST desde distintos clientes y el acceso a servicios externos con FastAPI, aplicándolo a un escenario completo de aplicación. En la parte de seguridad se cubren los conceptos de autenticación y autorización en servicios REST, el uso de JWT, la securización de un servicio CRUD y el consumo de servicios REST securizados. 
Por último, se aborda el despliegue de servicios REST en contenedores Docker, incluyendo la preparación del servicio, el despliegue del contenedor y el acceso al servicio desde el entorno desplegado.

👉 Una formación práctica para aprender a crear, probar, documentar, securizar y desplegar servicios REST con Python y FastAPI.
