¿Qué es el constructor __init__?
QUÉ ES
def __init__(self, nombre, edad):
Es el método que Python llama automáticamente cuando creas un objeto. Es el "molde" que define qué datos necesita cada instancia para existir. self siempre es el primer parámetro y representa al objeto que se está creando.
SELF
self.nombre = nombre
self es el objeto mismo. Al escribir self.nombre = nombre, estás guardando el valor recibido como atributo permanente del objeto. Sin self. sería una variable local que desaparece.
INSTANCIAR
persona1 = Persona("Ana", 25)
Al llamar Persona("Ana", 25), Python crea un nuevo objeto en memoria y llama a __init__ pasando el objeto como self y los argumentos en orden.
Paso a paso: crea y usa un objeto
PASO 1 — Definir la clase
class Persona: def __init__(self, nombre, edad): self.nombre = nombre self.edad = edad def __str__(self): return f"Persona: {self.nombre}, {self.edad} años" def saludar(self): return f"Hola, soy {self.nombre}"
PASO 2 — Crear instancias
p1 = Persona("Ana", 25) p2 = Persona("Luis", 30) print(p1 is p2) # False
PASO 3 — Usar atributos y métodos
print(p1.nombre) # "Ana" print(p1.saludar()) # "Hola, soy Ana" print(p1) # "Persona: Ana, 25 años"
