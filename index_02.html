Introducción

Hoy escribimos código que hace algo real. Vamos a trabajar con variables, tipos de datos (int, float, str, bool), operadores aritméticos y lógicos, y strings con format. La forma en que escribas esto hoy va a estar en cada cuaderno del resto del curso.

Contenidos de la sesión

1. Tipos primitivos

Cada dato en Python tiene un tipo: entero (int), decimal (float), texto (str) o booleano (bool). Python infiere el tipo automáticamente, pero es bueno saberlo. Usamos type() para verificar.

2. Operadores aritméticos

+ suma, - resta, * multiplica, / divide (devuelve float), // divide entero, % módulo (residuo), ** potencia.

3. Operadores de comparación

== igualdad (¿son iguales?), != desigualdad, < menor, > mayor, <= menor o igual, >= mayor o igual. Devuelven True o False.

4. Operadores lógicos

and (ambas condiciones verdaderas), or (al menos una verdadera), not (niega). Útiles para combinar comparaciones.

5. Strings y slicing

Indexación (acceso a caracteres), slicing (extraer partes), métodos (.upper(), .lower(), .replace(), etc.). Format con f-strings: f"Hola {nombre}".

6. Conversión de tipos

int(), float(), str() convierten entre tipos. Importante: input() devuelve texto, hay que convertir si quieres un número.
