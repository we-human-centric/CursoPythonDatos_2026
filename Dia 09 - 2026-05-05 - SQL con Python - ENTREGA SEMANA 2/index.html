Día 9 · SQL con Python — Pipeline completo + Entrega S2

Hoy juntan lo de toda la semana en un pipeline. Ingesta, limpieza, query, entrega. Y sí, aprenden SQL — no para convertirse en DBAs, sino para entender datos relacionales. Además, es el cierre de la Semana 2.

Contenidos de la sesión

Hoy cubrimos:

  • SQLite en Python: crear bases en memoria con sqlite3
  • Cargar DataFrames a SQL: df.to_sql()
  • Consultas SQL básicas: SELECT, WHERE, GROUP BY, JOIN
  • Ejecutar SQL desde Python: pd.read_sql_query()
  • SQL vs. Pandas: cuándo usar cada uno

Por qué SQL

SQL es el lenguaje de las bases de datos. En empresas, los datos viven en bases de datos relacionales. Si solo sabes Pandas, estás limitado. SQL te da acceso al verdadero pipeline de datos real. Además, entender joins es fundamental.

Antes de empezar

Necesitas:

  • Completado Día 8 (Estadística) — todo lo anterior
  • Entender que SQL es más explícito que Pandas: bueno para grandes datos
  • Coraje — SQL se ve feo al principio pero es lógico
